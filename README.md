# Task 3 - JWT Authentication Project

## Features
- User Signup
- User Login
- Password hashing using bcryptjs
- JWT token generation
- Protected routes in frontend
- Add / Edit / Delete products
- Logout functionality

## Tech Stack
- Frontend: React
- Backend: Node.js, Express
- Database: MongoDB

## Authentication Flow
1. User signs up
2. User logs in
3. Server generates JWT token
4. Token stored in localStorage
5. Protected pages accessible only with token

## How to run
### Backend
npm install
npm start

### Frontend
npm install
npm run dev
